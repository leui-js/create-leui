LeUI 可用于快速创建 OA 项目，响应式设计，内置 OA 登录态

## 功能特性

- 快速创建 Vue.js 3 项目
- 集成 Vuetify 框架，TypeScript 编写
- 集成登录、前端监控等通用组件
- 自动安装依赖

## 安装

全局安装 le-ui-cli：

```sh
npm install -g le-ui-cli 
```

## 创建项目

```sh
le-ui-cli create <my-project>
```
